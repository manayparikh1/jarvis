MARVIS: The Next Iron Man
MARVIS is an intelligent voice assistant that provides instantaneous, natural, and seamless experience. You talk, and it reacts. There is no wake word, no activation word, no listening for the system to “wake up.” From the very second you start speaking, MARVIS is already listening. Everything happens on your end, and all the assistant works from the one single HTML file.

Getting Started
In order to work with MARVIS, you will need to obtain a free Groq API key. Having done that, you should open index.html page right in Google Chrome. There is no setup and no server needed to get started. Just enter your API key, click “Start It Now” button, and MARVIS will be completely operational. To use MARVIS hands-free, just click AUTO button and start talking.

Modes
Talk Once  
This mode functions similarly to any push-to-talk voice assistants. You click the microphone icon, speak your query, receive the answer, and proceed in the same manner.

AUTO mode  
AUTO mode requires no button clicks. MARVIS continuously listens, responds to your queries after you stop talking. This mode is suitable for prolonged conversation, hands-free mode, or when you want the assistant to be responsive.

AUTO Mode
With AUTO mode, pressing buttons is not necessary at all. MARVIS will be listening and responding after you stop talking. This mode is useful for extended chats, hands‑free experience, and when you would like your virtual assistant to act more as a conversation partner.

Local Commands (Offline)
There are some preinstalled local commands that don't need any internet connection. They include asking about date and time, tossing coins, rolling dice, doing simple math operations, making a note in the memory, toggling between the day and night mode, and clearing the chat. These commands work without accessing the Groq API.

AI Chat
To enjoy full conversational capabilities, MARVIS uses llama‑3.3‑70b model via the Groq API. Several personalities are available and you can select one, such as sarcasm, professionalism, chaos, or pirate mode.

Voice Control
With MARVIS, you can control your experience by controlling its voice. You can select the voice, set the speaking speed, change the pitch, and set the volume level. Real-time visualization of your audio data is shown on screen in the form of a wave.

Memory
MARVIS can store information for you. To create a memo, say "Remember X", to recall memos, say "What do you remember?" or to reset all memos, say "Forget everything". Memos can also be added using the interface.

Quick Actions
There is a bunch of quick actions offered by MARVIS. It includes jokes, trivia, riddles, roasts, motivational quotes, coin flipping, dice rolling, time checking, and timers. All these quick actions are intended to be fast and easy to use without using menu options.

Extra Features
Apart from the mentioned options, MARVIS also has such extra features as exporting the chat to the file, night mode, managing API keys, clearing the chat, and stats bar.
Problems?
If your microphone isn’t working, you might have to start MARVIS locally on a server. One easy way to do that is:

Code
python3 -m http.server 8000
After that, use the website through the local server. Chrome is preferred since it offers the best voice recognition. Also, make sure your API key starts with gsk_, as per the requirements of Groq.

Privacy
All the processing takes place in your browser. MARVIS never transmits audio to any backend servers, and no data is stored. The assistant is completely self-contained.

Everything is contained within an HTML file. You can just open it, talk to it, and it answers back.
