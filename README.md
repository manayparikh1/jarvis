MARVIS: The Next Iron Man
No-nonsense AI voice assistant. Speak naturally and it responds. No wake word, no friction.

Getting Started
Obtain a free Groq API key

Open index.html in Chrome browser

Enter API key → START IT NOW!

Press AUTO and start speaking

Modes
Talk Once: Click on microphone icon, speak, respond, repeat.

AUTO: Continuous mode without button clicks

Local Commands Offline
Date, time, coin toss, dice roll, math calculations, memory notes, day/night mode, clear chat.

AI Chat
Using llama-3.3-70b with personalities: Sarcasm, Professionalism, Chaos, Pirate.

Voice Control
Select Voice, Speed, Pitch and Volume. Real-time waveform of your audio is displayed.

Memory
"Remember X", "What do you remember?", "Forget everything" or Manage notes manually.

Quick Actions
Joke, Interesting Fact, Riddle, Roast, Motivation, Coin, Dice, Time, Timer.

Extra Features
Chat Export, Night Mode, Change API Key, Clear Chat, Stats Bar.

Problems?
Mic not working? Set up a local server (python3 -m http.server 8000).
Use Google Chrome for voices. API key should start with gsk_.

Privacy
Everything happens in your browser. No backend, no data collection.

All in a single HTML file. Open it and speak.
